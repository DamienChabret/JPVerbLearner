Conjugaison des verbes

Les verbes se composent en 3 Groupes

  • Groupe 1 - Godan
  • Groupe 2 - ...
  • Groupe 3 - Irrégulier

Groupe 1

Les infinitifs des verbes se finissent toujours en -u. Conjugués, finisnet toujours en -i

  • う -> い ます
  • く -> き ます
  • ぐ -> ぎ ます
  • す -> し ます
  • つ -> ち ます
  • ぬ -> に ます
  • ぶ -> び ます
  • る -> り ます
  • ぬ -> に ます

Groupe 2

Terminent par -ru. Un verbe fait parti du deuxième groupe si la -ru est précédé par la voyelle -i ou -e. On enlève alors -ru et ajoute -masu. Exemple : - taberu -> tabemasu - okiru -> Okimasu
